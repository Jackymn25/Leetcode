两数之和（Two Sum） 解题思路

题意

给定整数数组 nums 与目标值 target，找出两个下标 i, j，使得 nums[i] + nums[j] == target（每个元素最多使用一次，且保证恰好一组解）。

思路

  1. 维护映射 值 → 下标 的表 idx。
  2. 遍历元素 x = nums[i]，计算补数 need = target - x。
  3. 若 idx 中已有 need，返回 [idx[need], i]；否则记录 idx[x] = i。
正确性第二次遇到配对值时，第一次的下标已被记录，立即返回即可。
复杂度查找/插入均摊 O(1)，整体时间 O(n)，额外空间 O(n)。

实现（C++）

#include <vector>
#include <unordered_map>
#include <iostream>

class Solution {
public:
    std::vector<int> twoSum(std::vector<int>& nums, int target) {
        std::unordered_map<int, int> idx; // 值 → 下标
        for (int i = 0; i < (int)nums.size(); ++i) {
            int need = target - nums[i];
            if (idx.count(need)) return { idx[need], i };
            idx[nums[i]] = i;
        }
        return {};
    }
};

int main() {
    std::vector<int> nums = {2, 7, 11, 15};
    int target = 9;
    auto ans = Solution().twoSum(nums, target);
    std::cout << ans[0] << " " << ans[1] << "\n";
    return 0;
}
